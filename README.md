StayEase is a React-based hotel discovery and booking frontend that allows users to browse hotels, explore destinations, and initiate bookings through a clean, multi-page interface. The application focuses on structured client-side routing, dynamic filtering, and authentication-based access control.

Key Features
1) Multi-page hotel discovery experience using React Router
2) Search and filter functionality to dynamically update hotel listings
3) Destination and attraction guide pages built with reusable components
4) Authentication-protected booking flow
5) Redirect-based access control for unauthenticated users

Authentication & Access Control

Booking actions are restricted to authenticated users.
When an unauthenticated user attempts to book a hotel, they are automatically redirected to the login page before proceeding.


Tech Stack :
1) React
2) React Router
3) JavaScript (ES6+)
4) Bootstrap
